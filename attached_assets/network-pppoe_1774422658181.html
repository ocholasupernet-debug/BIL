MikroTik PPPoE Server Script

Paste this in your MikroTik terminal to configure the PPPoE server with RADIUS authentication.

/interface pppoe-server server
add authentication=mschap2 default-profile=pppoe-default \
    interface=ether1-WAN max-mtu=1480 max-mru=1480 \
    mrru=disabled name=pppoe-in1 one-session-per-host=yes

/ppp profile
set *0 local-address=10.0.0.1 remote-address=pppoe-pool use-radius=yes

/ip pool
add name=pppoe-pool ranges=10.0.0.10-10.0.0.200

/radius
add address=127.0.0.1 secret=radiussecret service=ppp timeout=3000ms
