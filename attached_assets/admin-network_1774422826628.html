Hotspot Self-Install Script

Copy this script and paste it in your MikroTik terminal to set up the hotspot portal.

/ip hotspot profile add name=hsprof1 hotspot-address=192.168.1.1 dns-name=hotspot.example.com login-by=http-chap,http-pap use-radius=yes
/ip hotspot add name=hotspot1 interface=bridge1 profile=hsprof1 address-pool=hspool idle-timeout=none
/ip pool add name=hspool ranges=192.168.2.2-192.168.2.254
