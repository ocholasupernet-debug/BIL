Network — Self Install

Step-by-step guide to install the hotspot captive portal on a MikroTik router.

Prerequisites

MikroTik RouterOS 7.x or later

Check with: /system package print

WAN uplink connected on ether1

Active internet connection on the WAN port

RADIUS server running (optional but recommended)

FreeRADIUS or OcholaSupernet managed RADIUS

Installation Steps

1. Create IP Pool

Define the IP range to assign to hotspot users.

/ip pool add name=hs-pool ranges=192.168.10.10-192.168.10.200

2. Create Hotspot Profile

/ip hotspot profile
add dns-name=hotspot.ocholasupernet.com hotspot-address=192.168.10.1 \
    name=hsprof login-by=http-chap,http-pap use-radius=yes
3

3. Enable Hotspot on Interface

/ip hotspot
add address-pool=hs-pool interface=bridge1 name=hotspot1 \
    profile=hsprof idle-timeout=none keepalive-timeout=none
4

4. Add RADIUS Server

/radius add address=127.0.0.1 secret=radiussecret service=hotspot timeout=3000ms
5

5. Set Login Page URL (Custom Portal)

/ip hotspot profile set hsprof login-by=https login-page=/login.html

✓ After Setup

Clients connecting to your WiFi will be redirected to the hotspot login page. Register this router in the Routers section and set the portal URL to your OcholaSupernet hotspot login page.

Configure a PPPoE server on MikroTik for home broadband subscribers.

1

1. Create PPPoE Address Pool

/ip pool add name=pppoe-pool ranges=10.0.0.10-10.0.0.200
2

2. Set PPP Profile

/ppp profile set *0 local-address=10.0.0.1 remote-address=pppoe-pool \
    use-radius=yes dns-server=8.8.8.8,8.8.4.4
3

3. Enable PPPoE Server

/interface pppoe-server server
add authentication=mschap2 interface=ether1-WAN \
    name=pppoe-in1 one-session-per-host=yes \
    max-mtu=1480 max-mru=1480
4

4. Point to RADIUS

/radius add address=127.0.0.1 secret=radiussecret service=ppp timeout=3000ms

Install and configure FreeRADIUS on a Linux server alongside your OcholaSupernet instance.

1

1. Install FreeRADIUS

sudo apt update && sudo apt install freeradius freeradius-postgresql -y
2

2. Configure SQL Module

# /etc/freeradius/3.0/mods-available/sql
driver = "rlm_sql_postgresql"
dialect = "postgresql"
server = "localhost"
port = 5432
login = "radius"
password = "your-db-password"
radius_db = "radius"
3

3. Add MikroTik as NAS Client

# /etc/freeradius/3.0/clients.conf
client mikrotik-router {
    ipaddr  = 192.168.1.1
    secret  = radiussecret
    shortname = mikrotik01
}
4

4. Start Service

sudo systemctl enable freeradius && sudo systemctl start freeradius
sudo freeradius -X  # Test mode — check for errors
