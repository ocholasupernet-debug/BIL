MikroTik Pool Script

Copy this to create IP pools on your MikroTik router.

/ip pool add name=active-pool  ranges=192.168.1.10-192.168.1.200
/ip pool add name=expired-pool ranges=10.10.0.1-10.10.0.254
/ip pool add name=hotspot-pool ranges=192.168.10.10-192.168.10.200
/ip pool add name=pppoe-pool   ranges=10.0.0.10-10.0.0.200
